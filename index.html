(Antes conocido como iPython Notebook)

¿Qué es iPython?

Un interprete de Python ampliado

  • Comandos "mágicos"
  • coloreado de sintaxis
  • Autocompletado de código
  • Introspección
  • Mejores ayudas, documentación
  • debugging, etc...

iPython es Python con superpoderes

¿Qué es un notebook?

Una aplicación web, que permite ejecutar código a la vez que representar texto con formato, imágenes, diagramas y ecuaciones matemáticas como un todo

¿Para qué sirve?

  • Realización de análisis y estudios repetibles, verificables y modificables
