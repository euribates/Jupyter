¿Qué es iPython?

Un interprete de Python ampliado

  • Autocompletado de código

¿Qué es un notebook?

Una aplicación web, que permite ejecutar código incluyendo imágenes, diagramas y ecuaciones matemáticas de forma integrada

¿Para qué sirve?

  • Consola Python en web
  • Realización de análisis y estudios
  • Informes en vivo
  • Panel de mando

Son fáciles de compartir, repetibles, verificables y modificables
